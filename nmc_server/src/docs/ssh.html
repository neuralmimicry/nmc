SSH Key API Endpoints

Manage SSH public keys for secure access

← Back to API Home

Create SSH Key

Endpoint: POST /ssh/create

Creates a new SSH public key record with a name and optional description.

Request Body

{
    "name": "my-ssh-key-prod",
    "publicKey": "ssh-rsa AAAAB3NzaC1yc2EAAAADAQABAAABAQ...",
    "description": "SSH key for production servers"
}

Successful Response (HTTP 200)

{
    "success": true,
    "message": "SSH key 'my-ssh-key-prod' created successfully.",
    "data": {
        "id": "sshkey-abc123def456",
        "name": "my-ssh-key-prod",
        "publicKey": "REDACTED",
        "description": "SSH key for production servers"
    }
}

Error Responses

{
    "success": false,
    "message": "Missing required parameters: name or publicKey.",
    "data": {}
}
{
    "success": false,
    "message": "SSH key 'my-ssh-key-prod' already exists.",
    "data": {}
}

Get SSH Key

Endpoint: GET /ssh/get/{id}

Returns metadata for one SSH key by ID. The public key value is always redacted in API responses.

Path Parameters

  • id (string): The unique SSH key identifier.

Successful Response (HTTP 200)

{
    "success": true,
    "message": "SSH key 'sshkey-abc123def456' retrieved successfully.",
    "data": {
        "id": "sshkey-abc123def456",
        "name": "my-ssh-key-prod",
        "publicKey": "REDACTED",
        "description": "SSH key for production servers"
    }
}

Error Response (HTTP 404)

{
    "success": false,
    "message": "SSH key 'non-existent-id' not found.",
    "data": {}
}

List SSH Keys

Endpoint: GET /ssh/list

Lists registered SSH keys. Supports optional filtering by name.

Query Parameters

  • filter-name (optional, string): Filters keys whose names contain the provided text.

Successful Response (HTTP 200)

{
    "success": true,
    "message": "SSH keys listed successfully.",
    "data": [
        {
            "id": "sshkey-abc123def456",
            "name": "my-ssh-key-prod",
            "publicKey": "REDACTED",
            "description": "SSH key for production servers"
        },
        {
            "id": "sshkey-xyz789uvw012",
            "name": "dev-ssh-key",
            "publicKey": "REDACTED",
            "description": "Key for development instances"
        }
    ]
}

Delete SSH Key

Endpoint: DELETE /ssh/delete/{id}

Deletes a specific SSH key by ID.

Path Parameters

  • id (string): The unique SSH key identifier.

Successful Response (HTTP 200)

{
    "success": true,
    "message": "SSH key 'sshkey-abc123def456' deleted successfully.",
    "data": null
}

Error Response (HTTP 404)

{
    "success": false,
    "message": "SSH key 'non-existent-id' not found.",
    "data": {}
}
