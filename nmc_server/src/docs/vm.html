Create VM

Endpoint: POST /vm/create

Provisions a new virtual machine instance.

Tracey policy: Continuum-managed VMs must include a tracey block so the VM is tracked against a required Tracey agent.

Request Body:


{
    "name": "my-web-server-01",
    "sku": "nmx-h100-80",
    "region": "gb-mids",
    "osImage": "ubuntu-22.04",
    "publicKeyId": "sshkey-abc123def456",
    "initScript": "#!/bin/bash\nsudo apt update && sudo apt install -y nginx",
    "tracey": {
        "agent_id": "tracey-my-web-server-01",
        "status_addr": "http://10.0.12.44:48000"
    }
}

Successful Response (HTTP 200 OK):


{
    "success": true,
    "message": "VM 'my-web-server-01' created successfully.",
    "data": {
        "id": "vm-1a2b3c4d5e6f",
        "name": "my-web-server-01",
        "sku": "nmx-h100-80",
        "region": "gb-mids",
        "osImage": "ubuntu-22.04",
        "publicKeyId": "sshkey-abc123def456",
        "initScript": "#!/bin/bash\nsudo apt update && sudo apt install -y nginx",
        "status": "Creating",
        "tracey": {
            "required": true,
            "agent_id": "tracey-my-web-server-01",
            "status_addr": "http://10.0.12.44:48000",
            "compliance_state": "pending"
        }
    }
}

Error Response (HTTP 400 Bad Request if parameters missing):


{
    "success": false,
    "message": "Missing required parameters: name, sku, region, osImage, or publicKeyId.",
    "data": {}
}

Error Response (HTTP 409 Conflict if VM with name already exists):


{
    "success": false,
    "message": "VM 'my-web-server-01' already exists.",
    "data": {}
}

Error Response (HTTP 400 Bad Request if publicKeyId is invalid):


{
    "success": false,
    "message": "Invalid publicKeyId: 'non-existent-sshkey' not found.",
    "data": {}
}

Delete VM

Endpoint: DELETE /vm/delete/{id}

Deletes a specific virtual machine by its ID.

Path Parameters:

  • id (string): The unique ID of the VM to delete.

Successful Response (HTTP 200 OK):


{
    "success": true,
    "message": "VM 'vm-1a2b3c4d5e6f' deleted successfully.",
    "data": null
}

Error Response (HTTP 404 Not Found):


{
    "success": false,
    "message": "VM 'non-existent-id' not found.",
    "data": {}
}

List VMs

Endpoint: GET /vm/list

Lists all virtual machine instances. Supports optional filtering by name.

Query Parameters:

  • filter-name (optional, string): Filters VMs whose names contain the provided string.

Successful Response (HTTP 200 OK):


{
    "success": true,
    "message": "VMs listed successfully.",
    "data": [

            "id": "vm-1a2b3c4d5e6f",

            "publicKeyId": "sshkey-abc123def456",

            "status": "Running"
        },
        {
            "id": "vm-xyz789uvw012",
