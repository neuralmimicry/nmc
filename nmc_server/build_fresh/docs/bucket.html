Bucket API Endpoints

Create Bucket

Endpoint: POST /bucket/create

Creates a new storage bucket.

Request Body:


{
    "name": "my-new-bucket",
    "location": "eu-central",
    "type": "standard"
}

Successful Response (HTTP 200 OK):


{
    "success": true,
    "message": "Bucket 'my-new-bucket' created successfully.",
    "data": {
        "name": "my-new-bucket",
        "location": "eu-central",
        "type": "standard",
        "status": "Active"
    }
}

Error Response (HTTP 400 Bad Request if parameters missing):


{
    "success": false,
    "message": "Missing required parameters: name, location, or type.",
    "data": {}
}

Error Response (HTTP 409 Conflict if bucket exists):


{
    "success": false,
    "message": "Bucket 'my-new-bucket' already exists.",
    "data": {}
}

Delete Bucket

Endpoint: DELETE /bucket/delete/{name}

Deletes a specific storage bucket by its name.

Path Parameters:

  • name (string): The name of the bucket to delete.

Successful Response (HTTP 200 OK):


{
    "success": true,
    "message": "Bucket 'my-old-bucket' deleted successfully.",
    "data": null
}

Error Response (HTTP 404 Not Found):


{
    "success": false,
    "message": "Bucket 'non-existent-bucket' not found.",
    "data": {}
}
