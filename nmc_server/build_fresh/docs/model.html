Upload Model

Endpoint: POST /model/upload

Uploads a new machine learning model to a specified registry. In this server, this operation simulates an upload and acknowledges receipt of the model details.

Request Body:


{
    "filePath": "/path/to/my/model.onnx",
    "sku": "my-custom-model-v1",
    "registryName": "production-registry",
    "tag": "latest"
}

Successful Response (HTTP 200 OK):


{
    "success": true,
    "message": "Model 'my-custom-model-v1' version 'latest' uploaded to registry 'production-registry' successfully.",
    "data": {
        "filePath": "/path/to/my/model.onnx",
        "sku": "my-custom-model-v1",
        "registryName": "production-registry",
        "tag": "latest"
    }
}

Error Response (HTTP 400 Bad Request if parameters missing):


{
    "success": false,
    "message": "Missing required parameters: filePath, sku, registryName, or tag.",
    "data": {}
}

Error Response (HTTP 400 Bad Request for invalid JSON):


{
    "success": false,
    "message": "Invalid JSON body: [error details]",
    "data": {}
}

Error Response (HTTP 500 Internal Server Error for unexpected issues):


{
    "success": false,
    "message": "Server error: [error details]",
    "data": {}
}
